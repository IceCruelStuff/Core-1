### Features
- Blank Join/Leave/Death messages
- AutoSpawn onJoin()
- GMC, GMS, GMA, GMSPC, Day, & Night commands with permission-only access
- Hub command accessible by everyone
- Inventory clearing on Join, Leave, and Hub events
- Rules and Info commands
- Per-World Chat
